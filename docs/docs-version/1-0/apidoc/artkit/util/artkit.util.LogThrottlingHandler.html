artkit.util.LogThrottlingHandler#

class artkit.util.LogThrottlingHandler(handler, interval, max_messages)[source]#

A logging handler that limits the number of messages during a given time interval.

This handler can be used with Python’s native logging module to throttle the maximum number of messages sent during a given time interval. This is useful for preventing log spamming in high-throughput applications.

To follow the native logging flow (https://docs.python.org/3/howto/logging.html#logging-flow), functionality is implemented across the filter() and emit() methods.

Example:

import logging
from artkit.util import LogThrottlingHandler

console_handler = logging.StreamHandler()
throttling_handler = LogThrottlingHandler(
    console_handler,
    interval=5,
    max_messages=5
)
logger = logging.getLogger()
logger.addHandler(throttling_handler)
Bases:

Handler

Parameters:

  • handler (Handler) – the handler to wrap

  • interval (float) – the minimum interval in seconds between log messages with the same message

  • max_messages (int) – the maximum number of messages to log within the interval
