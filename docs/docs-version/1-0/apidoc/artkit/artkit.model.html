artkit.model#

Gen AI model endpoints.

This module has dedicated submodules for different types of Gen AI models:

  • artkit.model.llm: Large Language Models (text-to-text)

  • artkit.model.diffusion: Diffusion Models (text-to-image)

  • artkit.model.vision: Vision Models (image-to-text)

Submodules#

artkit.model.base

Base classes for the LLM module.

artkit.model.cache

Caching for ARTKIT model connections.

artkit.model.diffusion

Diffusion model implementations (text-to-image generation).

artkit.model.llm

Simple LLM adapter classes.

artkit.model.util

Utilities making use of the model classes.

artkit.model.vision

Vision model implementations (image-to-text generation).
